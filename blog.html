Question-1: What are the different ways to select an element in the DOM?

We can select DOM elements by their id (getElementById), class (getElementsByClassName), tag name (getElementsByTagName), or using CSS selectors (querySelector, querySelectorAll)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML can adds both html and text because it parses html tags. InnerText adds only visible text and ignores hidden elements. TextContent adds all text, including hidden elements, but doesn't parse html.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element instead of multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in DOM where an Event bubbling starts from the target element and moves upward to its parent.

Question-5: How do you create, add, and remove elements using JavaScript?

By using document.createElement() i can create an element , using appendChild() i can add element and using removeChild() i can remove an element.
